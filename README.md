|------|-------------|
| [Feedback-Synthesis](./feedback-synthesis) | Local AI-powered tool for synthesizing user feedback from Obsidian vaults using Claude |

## Getting Started

Each tool lives in its own sub-directory as an independent repository. Clone with submodules to get everything:

```bash
git clone --recurse-submodules https://github.com/rmcendarfer2017/Awesome-PM-Tools.git
```

Or if you already cloned without submodules:

```bash
git submodule update --init --recursive
```
